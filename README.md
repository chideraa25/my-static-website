# My Static Website

A beautiful responsive website made with HTML and CSS.

## Live Demo
http://dera-static-website.s3-website-us-east-1.amazonaws.com

## Features
- Fully responsive (works on phone and computer)
- Clean and modern design
- Built only with HTML and CSS
- Hosted on AWS S3

## How to Run Locally
1. Download or clone this repository
2. Open the `index.html` file in your browser
